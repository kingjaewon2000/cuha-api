# 쿠하 동아리

## 프로젝트 소개
쿠하 동아리는 함께 성장을 목표로하는 보안 동아리입니다.

코로나 시대에 오프라인으로 동아리 활동이 불가능해져 온라인으로 동아리 활동을 진행 하기 위해 프로젝트를 진행하였습니다.
